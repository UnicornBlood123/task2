2. При первом запуске потребуется установка необходимых модулей (npm install). Для запуска приложения используется команда "npm start".
3. Глобальная директория содержит в себе файлы настроки проекта. В директории "src" содежатся основные файлы разработки. 
Файл index.js служит для подключения остальных .js файлов и файла стилей style.css. Также файл index.html содержит основную разметку.
В директории "tests" содержатся два .js файла с тестированием математических функций.
